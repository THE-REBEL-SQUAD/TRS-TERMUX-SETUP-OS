​🛠️ Installation & Usage
​Termux open korun ebong nicher command gulo step-by-step copy-paste korun:
# Update and Upgrade System
pkg update && pkg upgrade -y

# Install Python and Git
pkg install python git -y

# Clone the Repository
git clone [https://github.com/THE-REBEL-SQUAD/TRS-TERMUX-SETUP-OS](https://github.com/THE-REBEL-SQUAD/TRS-TERMUX-SETUP-OS)

# Enter the Directory
cd TRS-TERMUX-SETUP-OS

# Run the Setup OS
python TRS-OS.py
